Text Obfuscation Mode marked with * may be nominally slower (each word is character matched 2x vs 1x, extra steps, etc)

Minimum Word Length To preserve short words like 'the' enter 3

Host Exclusions Regex Literally 
window.location.hostname
where the resulting string is inserted /^(HERE)$/i and the use of alternation | may allow listing multiples. Auto configured with button in popup on each site visited.

You may hit mulitple subdomains using something like 
[\w\.]*google.com|[\w\.]*facebook.com
You can test with the function itself! 

	
siteMatched('google.com', '[\w\.]*google.com|[\w\.]*facebook.com')

    
    

Host Enable Regex
